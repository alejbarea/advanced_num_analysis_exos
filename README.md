# Numerical Linear Algebra Exercises Workspace

This workspace is prepared for advanced numerical analysis and numerical linear algebra practice.

## Structure

- `src/`: reusable utility functions
- `exercises/`: exercise scripts
- `tests/`: pytest tests
- `notebooks/`: Jupyter notebooks
- `data/`: datasets used in exercises

## Setup

1. Create and activate a Python virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

Run the first exercise:

```bash
python exercises/ex01_linear_systems.py
```

Run tests:

```bash
pytest
```
